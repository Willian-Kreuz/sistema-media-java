# Sistema de Cadastro de Alunos

## 📌 Descrição
Aplicação em Java que cadastra alunos, armazena suas notas e calcula a média, informando a situação (Aprovado, Exame ou Reprovado).

## 🚀 Funcionalidades
- Cadastro de alunos
- Entrada de múltiplas notas
- Cálculo automático da média
- Verificação de situação

## 🛠️ Tecnologias
- Java
- ArrayList
- Orientação a Objetos

## ▶️ Como executar
1. Compile o projeto
2. Execute a classe Program
3. Siga as instruções no terminal
